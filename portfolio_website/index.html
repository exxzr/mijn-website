Netwerkbeheerder

Ik ben Eren Ozer, student Systeem- en Netwerkbeheer met een passie voor het efficiënt beheren en optimaliseren van netwerken en servers. Tijdens mijn studie ontwikkel ik sterke technische vaardigheden in Windows Server en Linux, gecombineerd met een analytische en probleemoplossende aanpak om systemen betrouwbaar, veilig en performant te houden.

Mijn focus ligt op Windows Server en Linux systeembeheer. Met Windows Server heb ik ervaring in user management, groeppenbeleid, Active Directory en serverbeheer via commandline en GUI. In Linux ben ik bedreven in het werken met commandline tools, shell scripting, gebruikersbeheer, permissies en systeemautomatisering. Deze combinatie geeft mij een breed inzicht in verschillende IT-omgevingen.

Ik benadruk problemen methodisch en stap voor stap. Bij een systeemprobleem analyseer ik eerst de oorzaak, onderzoek de beschikbare resources en implementeer een efficiënte oplossing. Dit probleemoplossende mindset pas ik toe op alles wat ik doe, van netwerkdiagnose tot scriptoptimalisatie.

Ik werk gestructureerd en gedisciplineerd. Ik documenteer mijn werk, test gegrond en zorg dat mijn implementaties voldoen aan best practices. Ik leer snel bij, ben gemotiveerd om nieuwe technologieën onder de knie te krijgen, en hou van uitdagingen die mij laten groeien.

Op deze portfolio vind je mijn projecten die mijn kennis en vaardigheden demonstreren. Van Linux scripting tot Windows Server beheer, van embedded systems tot webontwikkeling en professionele communicatie. Elk project toont mijn vermogen om complex systemen op te zetten, te beheren en problemen op te lossen.

Ik ben enthousiast over IT-infrastructuur en zie mezelf als iemand die bijdraagt aan solide, veilige en efficiënte systemen. Ben je benieuwd naar mijn werk? Bekijk mijn projecten en neem contact met me op!
