Gestructureerde Bekabeling

Projectbeschrijving

Voor deze opdracht hebben Benyamine en ik een nieuw netwerkontwerp gemaakt voor het bedrijf Webwiz. Het eerdere ontwerp hield geen rekening met gestructureerde bekabeling, wat kan leiden tot slechte overzichtelijkheid, foutgevoeligheid en beperkte uitbreidbaarheid.


Wij hebben een toekomstbestendig en professioneel netwerk uitgewerkt dat werkt met patchpanelen, racks, outlets en een duidelijke structuur volgens de principes van gestructureerde bekabeling.

Projectinformatie

Webwiz bestaat uit twee verdiepingen met 24 werkplekken, 20 desktops en 10 laptops. Elke werkplek heeft een VoIP-telefoon en elk kantoor bevat een MFP. Daarnaast zijn er twee on-prem servers in een beveiligde ruimte.


Ons ontwerp voorziet een centraal rack, gestructureerde bekabeling via outlets, patchpanelen, goede Wi-Fi ondersteuning en een netwerk dat eenvoudig uitbreidbaar en beheersbaar is.

Serverbeheer

Projectbeschrijving

Voor deze opdracht heb ik meerdere virtuele machines (VMs) opgezet in VMware Workstation Pro met Windows Server als centrale server en verschillende client machines. Het doel was een realistisch netwerk simuleren met een werkende server-client architectuur. Ik heb de server geconfigureerd met essentiële diensten en de clients verbonden met het netwerk. Dit project gaf mij inzicht in serveradministratie, netwerkbeheer en het beheren van meerdere machines vanuit een centrale locatie.

Projectinformatie

Dit project omvat meerdere virtuele machines geïmplementeerd in VMware Workstation Pro. De centrale Windows Server fungeert als de server met de nodige services, terwijl verschillende client machines aan het netwerk zijn gekoppeld. Het project behandelt de basis van serverbeheer, VM-management en netwerkconfiguratie in een virtuele omgeving. Door deze setup kunnen alle machines onderling communiceren en kan de server centraal beheerd worden.

Webtechnologie

Projectbeschrijving

Embedded

Projectbeschrijving

Voor deze opdracht heb ik een slotmachine gebouwd met een Arduino Uno, volledig zelf gecodeerd in de Arduino-taal. De slotmachine draait drie wielen met cijfers, waarbij de snelheid van de wielen verstelbaar is. Als drie gelijke cijfers verschijnen, speelt een overwinningsgeluid af. Bij een verlies hoort de gebruiker een verliesgeluid. Dit project gaf mij praktische ervaring met microcontrollers, hardware-programmering en het integreren van sensoren, displays en audio-output in een interactief systeem.

Projectinformatie

Dit project maakt gebruik van een Arduino Uno microcontroller geprogrammeerd in de Arduino-taal. De slotmachine bevat drie draaiende wielen met randomized cijfers, een instelbare snelheid, en audio-feedback via speakers. Het demonstreert kennis van digitale I/O, timing, randomisatie en audio-control met een embedded system. Een leuk en interactief project dat de basis van embedded systems en hardware-programmering toont.

Scripting

Projectbeschrijving

Voor deze opdracht heb ik met Linux gewerkt en verschillende systeembeheertaken uitgevoerd via commandline. Ik heb gebruikers aangemaakt, hun permissies beheerd, en shell scripts geschreven voor geautomatiseerde taken. Ik leerde hoe je met chmod en chown gebruikers- en groepsrechten toewijst en verwijdert, en hoe je scripts schrijft voor repetitieve processen. Dit project gaf mij waardevolle inzicht in Linux systeembeheer, scriptautomatisering en veiligheid door juiste rechtenbeheer.

Projectinformatie

Dit project behandelt Linux commandline operaties en shell scripting. Ik heb ervaring opgedaan met user management, file permissions (chmod/chown), directory structures en het schrijven van Bash scripts voor geautomatiseerde systeemtaken. Het project demonstreert fundamentele Linux systeembeheer vaardigheden die essentieel zijn voor systeemadministratoren en developers. Scripts kunnen gebruikt worden voor backup, logging, monitoring en andere repetitieve taken.
